#Blog Platform with Comments

A full-stack blogging website developed using Flask, SQLite, HTML, CSS, and Bootstrap. Users can register, log in, create blog posts, edit or delete posts, and interact through comments.

## Features
User Registration and Login
Create, Edit, Delete Blog Posts
Comment Section
User Profile Display
Logout Functionality
Responsive and Attractive UI
SQLite Database Integration
## Technologies Used
Python Flask
SQLite
HTML
CSS
Bootstrap
## How to Run
Open project in VS Code
Activate virtual environment
Install required packages
## Run:
python app.py
## Project Outcome
This project helped in understanding full-stack web development, database integration, authentication, CRUD operations, and user interaction features.
